<objective>
Genericize the workflow execution queue into a shared execution queue with job_type discriminator, add database-level dedup for identity match jobs, and add IDENTITY_MATCH to the SourceType enum.

Purpose: The generic queue is the #1 prerequisite for identity resolution (locked decision). All matching code in later phases depends on this queue infrastructure. The dedup index prevents race conditions that application-level checks cannot solve.

Output: Renamed `execution_queue` table with `job_type` column, dedup partial unique index, updated JPA entity/repository/model, new `ExecutionJobType` enum, `IDENTITY_MATCH` added to `SourceType`.
</objective>
